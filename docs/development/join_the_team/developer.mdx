## Obtained by

  - Submitting an application.
  - Frequently committing to our game, web, or discord bot repositories.
  - Granted by development team as code is accepted.
  
## Responsibilities

  - Writes code to complete issues from our [task board](https://github.com/orgs/SitekickRemastered/projects/1).

## Notes
  
  - Developers are generally granted moderation access, although none have it by default.
  - You must apply for this role and use our private repository if you wish to work on the game or server.
  - Our Discord bots are completely open source. Anyone can contribute code to them.
  - Intermediate understanding of C#, SQL, or Java is required.
  - Can not create codes or spawn items.

#### Website
- Written in Typescript, HTML, CSS, Markdown.

#### Game + Server
- Unity (C#).
- In-house game server (C#).
- MySQL 8 Database.

#### Discord Bots
- Written in Java.
- Relies on [Java Discord API (JDA)](https://github.com/discord-jda/JDA).
- Uses [Amazon Corretto 21 (OpenJDK)](https://docs.aws.amazon.com/corretto/latest/corretto-21-ug/downloads-list.html).
