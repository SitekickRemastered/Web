# Chipendium

<div className="row wiki-parent">
    <div className="col wiki-content-col">
The Chipendium is a book of Chips. When players obtain a chip for the first time, their Chipendium is updated so they can view the chip's information - even if they no longer hold the chip in question.

Each Chip has its own description that can only be read in the Chipendium. It also displays each chip's rarity, collection date, origin, how many are in circulation, and who the first player was to obtain it.
