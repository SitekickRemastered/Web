# Chipendium

<div className="row wiki-parent">
    <div className="col wiki-content-col">
:::warning	
This article is missing information and will be expanded in the future. 
:::
The Chipendium is a list of Chips. The player collects Chips to fill it up.

Each Chip has its own description that can only be read in the Chipendium. It also displays its rarity, type, and the date it was first obtained.

:::frantic
	Chips 0001-1000 are re-releases from the original [YTV Sitekick](/docs/ytv/sitekick/).
:::
